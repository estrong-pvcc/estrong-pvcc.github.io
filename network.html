Networking & the Internet

TERMS DEFINITIONS
Networking Classifications The way a computer network is classified ranging from small, personal networks to global wide area networks and the internet.
    LAN: Local Area Network
    MAN: Metropolitan Area Network
    WAN: Wide Area Network
Network Models Model representing objects and their relationships.
    Bus Network: Machines are all connected to a common communication line.
    Star Network: One machine serves as a central focal point to which all the others are connected.
Protocols Rules by which activities are conducted.
    CSMA/CD Ethernet: The right to transmit messages is controlled by the protocol.
    CSMA/CA Wifi: Wireless networks adopt the policy of trying to avoid collisions rather than trying to detect them.
Combining Networks Connecting the networks to form a larger version of the same “type” of network.
    Repeater: a device that passes signals back and forth between the two original buses (usually with some form of amplification) without considering the meaning of the signals.
    Bridge:It looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side.
    Switch: Essentially a bridge with multiple connections, allowing it to connect several buses rather than just two.
    Router: The connection between networks to form an internet.
    Gateway: The “point” at which one network is linked to an internet.
Client/server Model A popular convention used for interprocess communication
    P2P: Involves processes that provide service to and receive service from each other.
    Distributed System: They consist of software units that execute as processes on different computers.
    Cloud Computing: Huge pools of shared computers on the network can be allocated for use by clients as needed.
Internet Addressing Network address and a host address.
    Tier 1: Consist of very high-speed, high-capacity, international WANs.
    Tier 2: More regional in scope and less potent in their capabilities.
    Tier 3: An independent Internet, sometimes called an intranet, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.
IP Address A unique identifying address to each computer in the system.
    ICANN: Nonprofit corporation established to coordinate the Internets operation (Internet Corporation for Assigned Names and Numbers). icann Link
    DNS: Internet-wide directory system (Domain Name System).
    TLD: Suffix for a domain name (Top-Level Domains).
    Country Code TLDs: two-letter TLDs for each specific country. World Standards Link
Internet Protocols Set of rules governing the format of data sent over the internet or other network
    FTP: Application for listing and copying files across the network (File Transfer Protocol)
    HTTP: Traditional network applications have come to be handled by webpage (Hyper Text Transfer Protocol).
    SMTP: A way that two computers on the network may interact when transmitting an email message from one host to the other (Simple Mail Transfer Protocol).
World Wide Web An information system on the internet which allows documents to be connected to other documents by hypertext links, enabling the user to search for information by moving from one document to another.
    URL: A document is given a unique address (Uniform Resource Locator).
    Client-side activities: Activities performed by a client (such as a browser).
    Server-side activities: Activities performed by a server (such as a webserver).
Internet four-layer software Internet software has four layers rather than three, each consisting of a collection of software routines rather than people and businesses.
    Application Layer: Where the message originates and recieves message.
    Transport Layer: Chops message into packets and collects packets.
    Network Layer: Assigns address and detects the package.
    Link Layer: Transfers packet and receives packet.
Internet is A global computer network providing a variety of information and communication facilities.
    Packet Switching Network: Grouping data into packets that are transmitted over a digital network.
    Fault Tolerant: The ability of a computer system to continue operating without interruption when one or more components fail.
    Redundant: Installs a second connection the runs on a different backbone then thier primary connection.
Cybersecurity The study of protecting computers, networks, and data from such attacks.
    Malware: Malicious software can be transferred to, and executed on, the computer itself, or it might attack the computer from a distance.
    Virus: software that infects a computer by inserting itself into programs that already reside in the machine.
    Worm: An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
    Trojan Horse: A program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim.
    Spyware: Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
    Phishing: A technique of obtaining information explicitly by simply asking for it.
    DDoS attack: The work of generating the attacks is distributed over many hosts (Distributed Denial of Service).
Protection Controlling vandalism over network connections.
    Firewall: Filter traffic passing through a point in the network.
    Spam filter: Firewalls designed to block unwanted email.
    Proxy server: Software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
    Antivirus software: Used to detect and remove the presence of known viruses and other infections.
    Cryptography: The study of sending and receiving messages securely in the presence of adversaries.
    Public key encryption: Involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
