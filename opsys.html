Operating Systems Technology

TERMS DEFINITIONS
Operating System The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
Job Queue Jobs residing in mass storage waiting to be executed.
FIFO First in, first out, the order in which jobs are executed. Objects are removed from the queue in the order in which they arrived.
Interactive Processing Allows a program being executed to carry a dialogue with the user through remote terminals.
Time-Sharing Operating systems designed to provide service to multiple users at the same time.
Multiprogramming When time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
Application Software Consists of programs for performing tasks particular to the machine's utilization.
System Software Performs those tasks that are common to computer systems in general.
Utility Software Programs for performing activities that are fundamental to computer installations but not included in the operating system.
Graphical User Intrface (GUI) Objects to be manipulated, such as files and programs, are represented pictorially on the display as icons.
Shell Older user interfaces.
Kernal The internal part of the operating system.
Device Driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager Coordinates the use of the machine's mass storage facilities.
Memory Manager Charged with the task of coordinating the machine's use of main memory.
Virtual Memory A large fictional memory space created by paging.
Boot Strapping Transfers the operating system from mass storage into main memory.
ROM Read-Only Memory. Memory that cannot be altered.
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling The technique of holding data for output at a later but more convenient time.
Super User (administrator) This person gains highly privileged access to the operating system by identifying him or herself as the administrator during the login procedure.
Multi-factor Authentication The requirement to use more than one piece of evidence to gain access. One solution to humans using weak passwords.
