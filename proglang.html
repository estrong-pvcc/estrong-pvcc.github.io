Programming Languages

TERMS DEFINITIONS
Machine Language Programs for modern computers consist of sequences of instructions that are encoded as numeric digits.
Debugging Process of locating and correcting errors.
Assemblers Programs created to convert mneumonic expressions into machine language instructions.
Assembly Language A mneumonic system for representing programs.
Translator Translates programs that are expressed in high-level primitives into machine-language programs.
Compiler A translation program.
Interpreter An alternative to translators as another way to implement third-generation language.
Cross-Platform Software Type of application, program, or software that works on various operating systems or devices.
Natural Language English, German, and Latin.
Formal Language Programming Language.
Machine Independence Does not rely on the characteristics of a particular machine.
High-Level Programming Language A programming language that enables development of a program is user friendly programming context.
Object-Oriented Programming OOP Programming paradigm based on the concept of objects which can contain code and data.
Scripting Language A subset of the imperative program languages, it is the collection of languages
Data Types Variables that are initialized and updated, read, and written.
    Integer: Numeric data consisting of whole numbers, probably stored using twos' complement notation.
    Float: Numeric data that might contain values other than whole numbers, probably stored in floating-point notation.
    String: Sequence of characters.
Primitive Data Type int for integer and char for character
Data Structure Conceptual shape or arrangement of data.
Python Collection Data Structures
    List: Used to store multiple items in a single variable.
    Tuple: Used to store multiple items in a single variable.
    Set: Used to store multiple items in a single variable.
    Dictionary: Used to store data values in key: value pairs.
Function Set of instructions for performing a task that can be used as an abstract tool by other program units.
Local Variable A variable declared within a function.
Global Variable Variables whose scopes are not restricted to a particular part of a program.
Source Program Program in its original form before translation.
Object Program Program in its translated form.
